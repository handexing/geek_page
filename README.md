# geek_page
:triangular_flag_on_post: 极客屋前端界面。

## 项目介绍

极客屋是极客们的社区。在这里大家能够多多分享自己正在做的有趣事物、最近的学习、交流想法，在这里找到朋友认识更多有趣的人。

## 项目依赖
- mdui框架[官网](https://www.mdui.org/docs/)
- editor[官网](https://pandao.github.io/editor.md/)
- jquery
- jquery.cookie
- layui[官网](http://www.layui.com/)
